## Era 4 — The Atomic Age (2013–2018)

When modular thinking and reusable design became the foundation of the modern web.  
Interfaces became structured, consistent, and scalable.

### Core Technologies
 - TypeScript 1.0  
 - ECMAScript 2015 (ES6)  
 - Babel (transpilers & polyfills)  
 - React  
- Angular 2+  
- Vue  
- React Native & NativeScript  
- Polymer  
- Material Design  
- WebAssembly  
- Flow  
- CSS-in-JS  
- PostCSS  
- Autoprefixer  

### Browsers & Engines
- Google Chrome (V8 + Blink)  
- Mozilla Firefox (Quantum, SpiderMonkey)  
- Microsoft Edge (EdgeHTML)  
- Opera (Chromium / V8)

- Sublime Text
