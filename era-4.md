## Era 4 — The Atomic Age (2013–2018)

When modular thinking and reusable design became the foundation of the modern web.  
Interfaces became structured, consistent, and scalable.

### Technologies
- React  
- Angular 2+  
- Vue  
- Polymer  
- Material Design  

### Stack
- ECMAScript 2015 (ES6)  
- TypeScript 1.0, Flow, Babel  
- WebAssembly  

### Infrastructure
- GraphQL  
- Serverless Framework  
- Docker, Netlify, Vercel  
- AWS Lambda  
- Jenkins 2.x (Pipelines as Code)  
- Yarn, Lerna  

### Styling and UX
- PostCSS  
- Autoprefixer  
- CSS-in-JS  

### Browsers
- Google Chrome (V8 + Blink)  
- Mozilla Firefox (Quantum, SpiderMonkey)  
- Microsoft Edge (EdgeHTML)  
- Opera (Chromium / V8)  

### Influencers
- Dan Abramov  
- Jordan Walke  
- Evan You  

### IDE and Editors
- Sublime Text  
- Atom  
- Visual Studio Code  
- Brackets  
- Xcode  
- Pair Programming  

### Browser Debug Tools
- Chrome DevTools  
- Google Lighthouse  
- Chrome or Safari Remote Debugging for Android  
- React DevTools  
- Vue.js devtools  

### Culture & Workflow
- Component-Based Architecture  
- State Management (e.g., Flux/Redux, Vuex)  
- Declarative Programming
