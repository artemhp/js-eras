## Era 5 — The Progressive Era (2019–2024)

When front-end boundaries began to blur and developers took ownership across full-stack and performance.

### Core Technologies
- TypeScript (default)  
- React Server Components  
- Next.js  
- Remix  
- Svelte  
- Astro  
- WebAssembly (Wasm)  
- JS Tooling in Native (Rust, Go)  
- Deno  
- Tailwind CSS  
- Design tokens + Storybook  
- Babel macros / SWC / Vite transforms  

### Browsers & Engines
- Chromium dominance  
- Firefox Quantum / Servo experiments  
- Safari WebKit (PWA)  
- Arc / Brave / Vivaldi  
- Mobile parity  

### Tooling & IDEs
- Visual Studio Code  
- Online IDEs  
- Live Share  
- GitHub Copilot  
- ChatGPT integrations  
- Prettier  

### DevTools & Debugging
- React DevTools  
- Redux DevTools  
- Vue DevTools  
- Svelte Inspector  
- Playwright, Cypress  

### Infrastructure
- Vite  
- Serverless / Edge computing  
- Monorepo Tooling  
- Kubernetes  
- GitHub Actions, GitLab CI  
- Sentry, LogRocket, Datadog RUM  

### Culture & Workflow
- Meta-Frameworks  
- Islands Architecture  
- Jamstack  
- SSR & SSG as defaults  
- DX mindset  
- One codebase spans web, mobile, and TV surfaces
