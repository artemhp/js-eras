## Era 0 — Colonization of the Web (1991–1996)

When the web was a handful of linked documents and every page felt like unexplored territory.

### Core Technologies
- HTML 1.0 / HTML 2.0  
- Inline styling and `<font>` tags  
- Server-Side Includes (SSI)  
- CGI scripts (Perl, C)  

### Browsers & Engines
- WorldWideWeb / Nexus  
- NCSA Mosaic  
- Netscape Navigator 1.x  
- Lynx  
- Early Internet Explorer (Spyglass)  

### Tooling & IDEs
- Plain text editors (vi, Emacs, BBEdit)  
- Early WYSIWYG tools (HotDog, HoTMetaL)  
- Gopher/FTP clients for deployment  

### DevTools & Debugging
- “View Source”  
- Telnet logs and server error output  
- Validator.w3.org (beta)  

### Infrastructure
- CERN httpd  
- NCSA HTTPd  
- Gopher servers  
- Mailing lists and Usenet for support  

### Culture & Workflow
- Academic research pages and “home pages”  
- Under-construction GIFs and hit counters  
- Text-first design with inline images as accents  
- Manual link exchanges and web rings  

### Influencers
- Tim Berners-Lee  
- Marc Andreessen  
- Håkon Wium Lie (CSS proposal)  
- Brendan Eich (early JavaScript prototype groundwork)
